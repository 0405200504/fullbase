# Metodologia de Design FullBase

Ao realizar tarefas de design, siga obrigatoriamente as 4 fases documentadas em [EXECUTION_ROADMAP.md](../../docs/design/EXECUTION_ROADMAP.md).

## Princípios (Skills)
Consulte sempre o [DESIGN_SYSTEM_MANIFESTO.md](../../docs/design/DESIGN_SYSTEM_MANIFESTO.md) para garantir:
1. Branding e Identidade Futurista.
2. UX Estratégica (Esforço Zero).
3. UI Sofisticada (Microinterações premium).
4. Alinhamento com Visão de Produto.

## Passos para o Agente:
1. **Contextualizar Base de Design:** Verifique se o Moodboard e a Guia de Estilo foram estabelecidos.
2. **Wireframe Primeiro:** Nunca pule a validação de fluxo para ir direto ao visual, a menos que seja um ajuste menor.
3. **Componentização:** Todo elemento novo deve ser adicionado ao Design System (Tailwind tokens ou componentes React).
4. **Animação:** Use microinterações para dar o toque "premium".
